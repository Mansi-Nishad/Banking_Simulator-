## 📌 Overview

A desktop-based Banking Simulator that replicates real-world banking operations with secure authentication and role-based access control.

---

## ✨ Features

### 🔐 Authentication
- Role-based login system (Admin & Customer)
- Captcha verification for secure access

### 💳 Banking Operations
- Deposit funds
- Withdraw money
- Transfer funds between accounts

### 👨‍💼 Account Management
- Admin can create, view, and close accounts
- Customers can update personal details

### 📧 Email Integration
- OTP verification for password recovery
- Secure account closure via email
